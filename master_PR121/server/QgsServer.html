handleRequest(self, request: QgsServerRequest, response: QgsServerResponse, project: QgsProject = None)¶

Handles the request. The query string is normally read from environment but can be also passed in args and in this case overrides the environment variable

Parameters:

  • request (QgsServerRequest) – a QgsServerRequest holding request parameters

  • response (QgsServerResponse) – a QgsServerResponse for handling response I/O)

  • project (QgsProject = None) – a QgsProject or None, if it is None the project is created from the MAP param specified in request or from the QGIS_PROJECT_FILE setting
