Class: QgsLayoutView¶

class qgis.gui.QgsLayoutView¶

Bases: PyQt5.QtWidgets.QGraphicsView

A graphical widget to display and interact with QgsLayouts.

QgsLayoutView manages the layout interaction tools and mouse/key events.

New in version 3.0.
