class qgis.gui.QgsColorSliderWidget¶

Bases: QgsColorWidget

A composite horizontal color ramp widget and associated spinbox for manual value entry.

New in version 2.5.

QgsColorSliderWidget(parent: QWidget = None, component: QgsColorWidget.ColorComponent = QgsColorWidget.Red) Construct a new color slider widget.

Parameters:

  • parent – parent QWidget for the widget

  • component –
