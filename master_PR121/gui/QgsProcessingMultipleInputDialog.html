Class: QgsProcessingMultipleInputDialog¶

class qgis.gui.QgsProcessingMultipleInputDialog¶

Bases: PyQt5.QtWidgets.QDialog

A dialog for selection of multiple layer inputs.

Note

Not stable API

New in version 3.14.

QgsProcessingMultipleInputDialog(parameter: QgsProcessingParameterMultipleLayers, selectedOptions: Iterable[Any], modelSources: Iterable[QgsProcessingModelChildParameterSource], model: QgsProcessingModelAlgorithm = None, parent: QWidget = None, flags: Union[Qt.WindowFlags, Qt.WindowType] = Qt.WindowFlags()) Constructor for QgsProcessingMultipleInputDialog.

The selectedOptions list may contain extra options which are not present in availableOptions, in which case they will be also added as existing options within the dialog.
