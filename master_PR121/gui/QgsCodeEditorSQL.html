class qgis.gui.QgsCodeEditorSQL¶

Bases: QgsCodeEditor

A SQL editor based on QScintilla2. Adds syntax highlighting and code autocompletion.

Note

may not be available in Python bindings, depending on platform support

New in version 2.6.
