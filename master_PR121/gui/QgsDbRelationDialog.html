Class: QgsDbRelationDialog¶

class qgis.gui.QgsDbRelationDialog¶

Bases: PyQt5.QtWidgets.QDialog

A dialog for configuration of the properties of a relationship in a database.

New in version 3.30.

QgsDbRelationDialog(connection: QgsAbstractDatabaseProviderConnection, parent: QWidget = None, flags: Union[Qt.WindowFlags, Qt.WindowType] = Qt.WindowFlags()) Constructor for QgsDbRelationDialog, with the specified parent widget and window flags, for the specified connection.

Ownership of connection is transferred to the widget.
