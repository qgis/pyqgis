Class: QgsScaleVisibilityDialog¶

class qgis.gui.QgsScaleVisibilityDialog¶

Bases: PyQt5.QtWidgets.QDialog

A dialog allowing users to enter a scale visibility range.

QgsScaleVisibilityDialog(parent: QWidget = None, title: str = ‘’, mapCanvas: QgsMapCanvas = None) Constructor for QgsScaleVisibilityDialog, with specified dialog title. The mapCanvas argument can be used to associate the dialog with a map canvas, allowing use of the current map scale within the dialog.
