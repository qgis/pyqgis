class qgis.gui.QgsOrderByDialog¶

Bases: PyQt5.QtWidgets.QDialog

This is a dialog to build and manage a list of order by clauses.

New in version 2.14.

QgsOrderByDialog(layer: QgsVectorLayer, parent: QWidget = None) Create a new order by dialog. This helps building order by structures.

Parameters:

  • layer – The vector layer for which the order by should be produced

  • parent –
