Class: QgsMessageLogViewer¶

class qgis.gui.QgsMessageLogViewer¶

Bases: PyQt5.QtWidgets.QDialog

A generic dialog widget for displaying QGIS log messages.

QgsMessageLogViewer(parent: QWidget = None, fl: Union[Qt.WindowFlags, Qt.WindowType] = QgsGuiUtils.ModalDialogFlags) Create a new message log viewer. The viewer will automatically connect to the system’s QgsApplication.messageLog() instance.
