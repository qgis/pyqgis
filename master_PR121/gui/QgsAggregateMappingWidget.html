Class: QgsAggregateMappingWidget¶

class qgis.gui.QgsAggregateMappingWidget¶

Bases: QgsPanelWidget

The QgsAggregateMappingWidget class creates a mapping for defining sets of aggregates of fields from a QgsFields object.

New in version 3.14.

QgsAggregateMappingWidget(parent: QWidget = None, sourceFields: QgsFields = QgsFields()) Constructs a QgsAggregateMappingWidget from a set of sourceFields. A parent object can also be specified.
