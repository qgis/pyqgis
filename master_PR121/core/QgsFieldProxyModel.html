Class: QgsFieldProxyModel¶

class qgis.core.QgsFieldProxyModel¶

Bases: PyQt5.QtCore.QSortFilterProxyModel

The QgsFieldProxyModel class provides an easy to use model to display the list of fields of a layer.

New in version 2.3.

QgsFieldProxyModel(parent: QObject = None) QgsFieldProxModel creates a proxy model with a QgsFieldModel as source model. It can be used to filter the fields based on their types.
