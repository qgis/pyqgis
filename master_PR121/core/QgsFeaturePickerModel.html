Class: QgsFeaturePickerModel¶

class qgis.core.QgsFeaturePickerModel¶

Bases: QgsFeaturePickerModelBase

Provides a list of features based on filter conditions. Features are fetched asynchronously.

New in version 3.0.

QgsFeaturePickerModel(parent: QObject = None) Create a new QgsFeaturePickerModel, optionally specifying a parent.
