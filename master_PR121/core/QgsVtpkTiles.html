Class: QgsVtpkTiles¶

class qgis.core.QgsVtpkTiles¶

Bases: sip.wrapper

Utility class for reading and writing ESRI VTPK files.

New in version 3.26.

QgsVtpkTiles(filename: str) Constructs VTPK reader (but it does not open the file yet)
