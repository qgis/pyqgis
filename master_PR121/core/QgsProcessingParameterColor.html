Class: QgsProcessingParameterColor¶

class qgis.core.QgsProcessingParameterColor¶

Bases: QgsProcessingParameterDefinition

A color parameter for processing algorithms.

QgsProcessingParameterColor should be evaluated by calling QgsProcessingAlgorithm.parameterAsColor().

New in version 3.10.

QgsProcessingParameterColor(name: str, description: str = ‘’, defaultValue: Any = None, opacityEnabled: bool = True, optional: bool = False) Constructor for QgsProcessingParameterColor.

If opacityEnabled is True, then users will have the option of varying color opacity.
