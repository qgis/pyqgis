Class: QgsProcessingAlgRunnerTask¶

class qgis.core.QgsProcessingAlgRunnerTask¶

Bases: QgsTask

QgsTask task which runs a QgsProcessingAlgorithm in a background task.

New in version 3.0.

QgsProcessingAlgRunnerTask(algorithm: QgsProcessingAlgorithm, parameters: Dict[str, Any], context: QgsProcessingContext, feedback: QgsProcessingFeedback = None, flags: Union[QgsTask.Flags, QgsTask.Flag] = QgsTask.CanCancel) Constructor for QgsProcessingAlgRunnerTask. Takes an algorithm, algorithm parameters and processing context.

Since QGIS 3.26, the flags argument allows control over task flags.
