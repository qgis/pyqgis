Class: QgsScaleBarSettings¶

class qgis.core.QgsScaleBarSettings¶

Bases: sip.wrapper

The QgsScaleBarSettings class stores the appearance and layout settings for scalebar drawing with QgsScaleBarRenderer.

New in version 3.0.
