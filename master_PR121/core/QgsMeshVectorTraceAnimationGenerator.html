Class: QgsMeshVectorTraceAnimationGenerator¶

class qgis.core.QgsMeshVectorTraceAnimationGenerator¶

Bases: sip.wrapper

A wrapper for QgsMeshParticuleTracesField used to render the particles. Available for Python binding

New in version 3.12.
