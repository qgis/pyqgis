Class: QgsPointCloudRgbRenderer¶

class qgis.core.QgsPointCloudRgbRenderer¶

Bases: QgsPointCloudRenderer

An RGB renderer for 2d visualisation of point clouds using embedded red, green and blue attributes.

New in version 3.18.
